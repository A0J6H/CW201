# CW201
How to Run the server:
Make sure you have node and npm installed,
node -v
npm -v
If not then install them
Go into the server file using cd server
then run the server using the following command: 
node server.js
Then you should get the output that the server is running on the localhost:5000

Once the server is running you can run the index.html file:
To do this go into the index.html file then run without debugging.


NOTE: if you run the app and it adds extra files you have done it wrong. Please follow these steps thank you Andrew.
